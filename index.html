lEditor

A clean text editor

Usage

0. It uses jQuery

1. Include lEditor.js and lEditor.css

2. Bind $(a-div).lEditor(); to onload event

3. It's possible to add more options: $(a-div).lEditor({
    height: '300px',
    color: '#ff0'
});

Options

height
a css height string, the total height of the editor

font
initial text font, default is Arial

font_size
initial text font size, a number from 1 - 7, default is 4.

color
initial text color, default is #000

bg_color
initial text background color, default is #fff

toolbar
layout of button groups in the toolbar, default layout is as the top one. It accecpts an array of strings. possible strings:
undo (undo/redo),
font (name/size),
style (b/i/u),
color (fore/background),
align (left/center/right),
list (ul/ol),
link (add/remove),
insert (img, code)
